BPMN Ontology

Version: 1.01

Date(y-m-d): 2009-09-06

Authors: Chiara Ghidini, Marco Rospocher and Luciano Serafini

Description: The BPMN Ontology is a formalization in OWL-DL of the structural components of the Business Process Modelling Notation (BPMN), based on the latest stable BPMN specifications from OMG [BPMN Version 1.1 - January 2008]. The development of the ontology was guided by the description of the complete set of BPMN Element Attributes and Types contained in Annex B of the BPMN specifications.

Download: The BPMN Ontology. A textual description in terms of Description Logics is available here.

Contacts: For more information, please contact Marco Rospocher. If you find the ontology useful, or you are using in it in some project/application, or you have any comment about it, please let us know!
