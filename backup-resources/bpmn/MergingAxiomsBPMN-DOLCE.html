Merging Axioms BPMN - DOLCE

Version: 1.0

Date(y-m-d): 2009-06-11

Authors: Chiara Ghidini, Kamrul Hasan, Marco Rospocher, and Luciano Serafini

Description: The BPMN ontology provides a clear semantic formalisation of the structural components of BPMN and the DOLCE ontology is the first module of a Foundational Ontologies Library. In this paper we provide a textual description of the merging axioms (see here for more details on merging axioms) between BPMN ontology and DOLCE ontology based on Description Logic formalism.

Download: BPMN-DOLCE Merging Axioms Ontology. A textual description in terms of Description Logics is available here.
